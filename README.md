# GreenPrompt-Extension
GreenPrompt est une extension de navigateur conçue pour sensibiliser les utilisateurs à l'impact environnemental caché de l'IA générative. À chaque interaction sur ChatGPT, l'extension calcule en temps réel l'estimation de la consommation d'eau (refroidissement des serveurs) et l'empreinte carbone (CO2) générée, en affichant un feedback visuel.
